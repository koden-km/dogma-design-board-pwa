# Dogma Design Board PWA

DDD event storming board for Dogmatiq applications.

https://koden-km.github.io/dogma-design-board-pwa/

> [!NOTE]
> This is a work in progress proof of concept. There are many missing features.

## Dev

### Run dev server

```shell
npm install
npm run dev
```

### Run tests

**Once**

```shell
npm run test
```

**Continuously / Watch**

```shell
npm run testing
```
